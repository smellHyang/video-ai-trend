# 영상 AI 대시보드 빌드 스킬

dashboard-builder 에이전트가 사용한다. `analysis/report.json`과 `analysis/articles.json`을 읽고 `dashboard/index.html`을 생성한다.

## 대시보드 구성

```
┌─────────────────────────────────────────────┐
│  🎬 영상 AI 트렌드 모니터  [데이터 기준 날짜]  │
├──────────────────────────────────────────────┤
│  [이번 주 요약 카드 - 3줄 요약]               │
├──────────────────────────────────────────────┤
│  주요 트렌드 카드 (최대 5개, grid 2열)         │
├─────────────────┬────────────────────────────┤
│  기업별 활동     │  주목 기술 태그             │
│  (수평 바 차트) │  (클라우드 스타일)           │
├─────────────────┴────────────────────────────┤
│  에디터 픽: 이번 주 놓치면 안 될 뉴스 (최대5개) │
├──────────────────────────────────────────────┤
│  전체 기사 목록 (카테고리 필터 + 중요도 정렬)  │
└──────────────────────────────────────────────┘
```

## 기술 스택

- **HTML5 + CSS3** — CSS 변수로 테마 관리, flexbox/grid 레이아웃
- **Vanilla JS** — 외부 프레임워크 없음 (필터링, 정렬, 인터랙션)
- **Chart.js CDN** — `https://cdn.jsdelivr.net/npm/chart.js` (차트 시각화)
- **데이터 인라인** — JSON 데이터를 JS 변수로 HTML에 직접 삽입

## 디자인 원칙

- **다크 테마** — 배경 `#0f1117`, 카드 `#1a1d27`, 강조 `#6c63ff`
- **가독성** — 최소 폰트 14px, 라인 높이 1.6, 충분한 여백 (16px+)
- **색상 의미 일관성:**
  - importance high → `#ff6b6b` (빨강 계열)
  - importance medium → `#ffd93d` (노랑 계열)
  - importance low → `#6bcb77` (초록 계열)
  - category별 고유 색상 (차트 범례와 태그에 동일 적용)
- **반응형** — 768px 미만에서 단열 레이아웃

## 인터랙티브 기능

1. **카테고리 필터** — 전체 / generation / understanding / editing / multimodal / industry / research
2. **중요도 정렬** — high → medium → low 순 기본값, 날짜순 전환 가능
3. **기사 카드 클릭** — 원본 URL로 새 탭 열기 (URL 없으면 버튼 비활성화)
4. **데이터 업데이트 날짜 표시** — 헤더에 "데이터 기준: YYYY-MM-DD" 표시

## 생성 프로세스

1. `analysis/report.json` 읽기
2. `analysis/articles.json` 읽기
3. `dashboard/` 디렉토리 생성
4. 데이터를 JS 객체로 직렬화
5. HTML 템플릿에 데이터와 스타일/스크립트 인라인 삽입
6. `dashboard/index.html` 저장
7. 파일 크기 확인 (너무 크면 데이터 일부 트리밍)

## 에러 처리

- **데이터 파일 없음**: 각 섹션에 "데이터 수집 중..." 플레이스홀더 + 새로고침 안내
- **URL 없는 기사**: 링크 버튼 비활성화 + 시각적으로 구분

## 완료 알림

생성 완료 후 오케스트레이터에게 SendMessage:
```

기사 {N}개 표시, 트렌드 {N}개, 에디터 픽 {N}개
```
